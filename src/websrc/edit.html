Macro commands
    Pnn - Prepare for nn secs
    Wnn - Work for nn secs
    Rnn - Rest for nn secs
    Cnn - CountDown for nn secs
    Bnn - Beep for last nn secs
    Xn - eXchange control with action n
    (nn ....) - Repeat nn times
    [nn ....] - Repeat silent nn times

E.g.

    b3p10(3w30r20)

Beep for the last 3 seconds of each action, Prepare for 10 seconds, repeat the following 3 times - Work for 30 seconds then Rest for 20 seconds
